== Multi-prefixes und -suffixes

!!!This feature funktioniert im Moment nur mit PermissionsEx!!!

Wenn du das Multi-prefixes/suffixes feature aktiviert hast, bekommt der Spieler das Prefix von allen Gruppen in denen er ist.
Beispiel:

Du hast zweui Gruppen:
Owner und Moderator.

Das Prefix von Owner ist '&4Owner ' und das Prefix von Moderator ist '&9Moderator '.
Du bist in beiden Gruppen.
Dann wäre dein Prefix: '&4Owner &9Moderator '.
