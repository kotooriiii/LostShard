== Multi-prefixes et -suffixes

!!!Cette fonctionnalité est seulement disponible, si vous avez PermissionsEx installé!!!

Si vous avez Multi-prefixes/suffixes activé, le préfixe va s'appliquer au prefixes/suffixes de TOUS les groupes auxquels appartient le joueur.
Exemple:

Vous avez deux groupes:
Owner et Moderator.

Le préfixe de Owner est '&4Owner ' et le préfixe de Moderator est '&9Moderator '.
Vous êtes dans les groupes Owner ET Moderator.
Votre prefixe dans le chat va être: '&4Owner &9Moderator '.
Si maintenant vous avez un préfixe spécial pour vous, il va être aussi ajouté dans le préfixe du chat.
